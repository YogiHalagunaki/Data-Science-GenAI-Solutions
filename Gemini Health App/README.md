The **Gemini Health App** is a Streamlit-powered web application that uses **Google's Gemini API** (and optionally Azure OpenAI) to analyze food images and return nutritional information, calorie estimates, and health advice.

---

## ðŸš€ Features

- Upload a food image (JPG/PNG).
- Automatically detect food items.
- Get total calories and a per-item breakdown.
- See macronutrient ratios (carbs, fats, fibers, etc.).
- Determine if the food is healthy or not.
- Get a possible recipe based on the image.
